# LE Money Tracker

A small native Android app for tracking Egyptian pound spending and income.

## Features

- Add income and expense transactions in LE.
- Choose type, category, payment method, and account/wallet.
- Set a monthly budget.
- See monthly income, spending, remaining balance, and budget progress.
- Filter by month.
- Delete transactions.
- Data is stored locally on the phone with SQLite.

## Build APK

Open this folder in Android Studio, let Gradle sync, then choose:

`Build` -> `Build Bundle(s) / APK(s)` -> `Build APK(s)`

The debug APK will be created under:

`app/build/outputs/apk/debug/app-debug.apk`
